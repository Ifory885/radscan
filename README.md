# radscan
Rad浏览器爬虫结果筛选优化脚本
由于rad浏览器爬虫工具输出的结果包含很多图片、JS文件、css文件、字体等文件，显得有点杂乱无章。</br></br>
因此该脚本会对rad输出的结果进行自定义的文件后缀名、目录过滤及结果去重，还可以自动对输出的URL进行访问，获取其状态码、返回包长度和网站标题。</br>
rad.exe下载链接：https://stack.chaitin.com/tool/detail?id=2
